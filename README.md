# SHE-INTEL INDIA

A context-aware health intelligence platform for Indian women with 14 ML models, period tracking, and India-specific health context.

## Features
- 14 ML Models (99%+ accuracy)
- Period Tracking with Cycle Prediction
- Health Risk Analysis
- India-Specific Context (diet, schemes, AQI)
- Multi-language Support
- JWT Authentication

## Setup
```bash
cd backend
pip install -r requirements.txt
python main.py
```
Server runs at http://localhost:8001

## Deployment
Deploy to Railway for free tier - see Railway.app

## Tech Stack
- Frontend: HTML/CSS/JS
- Backend: Python FastAPI
- ML: scikit-learn, CatBoost
- Database: SQLite
