# SHE-INTEL INDIA - Frontend Specification

## Design Philosophy
- Premium dark aesthetic — NOT a generic AI chatbot
- Bloomberg Terminal meets luxury wellness app
- Rich, data-dense but organized interface
- No "typing indicator" or "chat bubbles"

## Color Palette
- Background Primary: #0A0A0F (near-black with blue undertone)
- Background Secondary: #12121A (card backgrounds)
- Background Tertiary: #1A1A24 (elevated elements)
- Accent Primary: #E85A5A (risk/alert - warm red)
- Accent Secondary: #5AE8B5 (safe/positive - mint green)
- Accent Tertiary: #E8C85A (warning - amber)
- Text Primary: #FAFAFA
- Text Secondary: #A0A0B0
- Text Muted: #606070
- Border: #2A2A35

## Typography
- Headlines: Playfair Display, 600 weight
- Body: DM Sans, 400/500 weight
- Monospace (data): JetBrains Mono

## Layout Structure
1. **Header** - Logo, tagline, state selector dropdown
2. **Hero Section** - Tagline + "Run Health Analysis" CTA
3. **Input Area** - Symptom text input (NOT a chatbot)
4. **Analysis Loader** - 5-step animated progress (rotating AI steps)
5. **Results Dashboard** - Multi-panel result display
6. **Footer** - Disclaimer, privacy links

## Key Visual Elements (NON-GENERIC)
- Animated circular risk gauge (SVG, animated stroke-dasharray)
- Vertical bar charts for community data
- "Food chips" for diet recommendations
- AQI badge with live color coding
- Staggered fade-in animations (not generic slide-up)
- Subtle grain texture overlay on background
- No chat bubbles — data cards only
